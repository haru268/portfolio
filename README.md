環境構築手順
1) .env（※必要な環境変数がある場合のみ）
   cp .env.example .env   ← ないならスキップ可
   ※ ブラウザから使う値は NEXT_PUBLIC_ を付ける。実運用は .env.local 推奨

2) パッケージインストール
   npm i

3) 開発サーバ起動
   npm run dev

4) ブラウザ
   http://localhost:3000/
